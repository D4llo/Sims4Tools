How to use s4pe

Open the package you want to edit; this will show you the list of resources on the left side and a preview of the selected resource on the right side. To edit a resource, hit the Grid button at the bottom of the window. Do not edit the game files directly ‒ copy the resources you want into a new package and edit them there.

To sort the resource list, use the “Tag”, “Type”, “Group” etc headings at the top.

To find resources by name, type, group and so on, type/paste what you are looking for in the Filter fields at the bottom of the window, then tick the box in front, hit Set and tick Filter Active. Note that the IDs are hexadecimal and need to be prefixed with “0x”.

To find resources by content (like tags or internal names), use the Search tool in the top menu: Tools > Search. String search is case sensitive.

To renumber, copy, export/import, duplicate .. a resource, use the contextual menu (rightclick on the resource in the list) or the Resource menu at the top. Export > To File always dumps the resource as-is. Options like Export to DDS (available on _IMG resources) are using external helpers that usually do some reformatting or conversion so that the resource will be editable in other tools.

To see a list of installed helpers or activate/deactivate them, use Settings > External Program Settings. Here you can also assign external programs for hex and text editing.

Troubleshooting

  • “Could not find a resource handler” message in the preview window and all options greyed out

    This means you have extracted s4pe with something other than 7zip, which is the single most common reason that s4pe is not working correctly. Download 7zip and try again.
